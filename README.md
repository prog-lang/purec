# `purec` - Pure Compiler

```text
Pure Compiler

Usage: purec [OPTIONS] <SOURCE>

Arguments:
  <SOURCE>  Path to source code file

Options:
  -o, --output <OUTPUT>  Path to output file [default: main.js]
      --arch <ARCH>      Output architecture (vm | node) [default: node]
  -h, --help             Print help
  -V, --version          Print version
```
